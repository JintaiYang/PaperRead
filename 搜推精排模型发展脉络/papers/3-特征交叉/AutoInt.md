# AutoInt

**AutoInt: Automatic Feature Interaction Learning via Self-Attentive Neural Networks**

Weiping Song, Chence Shi, Zhiping Xiao, et al. (北京大学, UCLA, Mila · CIKM 2019)

## 核心贡献

首次将 Multi-head Self-Attention 系统性地应用于特征交叉，实现输入自适应的动态高阶特征交互，并通过 attention 权重提供可解释性。为后续 Transformer-based 推荐模型（OneTrans、HyFormer 等）奠定概念基础。
